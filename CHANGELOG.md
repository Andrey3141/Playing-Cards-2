## [v1.2.0] - 2026-03-30

### Added
- Новые карты (id 11-13):
  - Курсовая (id=11): с вероятностью 90% убивает самую сильную карту противника, но сама умирает
  - Фантазер (id=12): с вероятностью 50% атака +20, но 40% шанс пропуск хода
  - Стример-неудачник (id=13): с вероятностью 40% атака +12, но 70% что это иллюзия
- Эффект иллюзии:
  - Анимация затемнения экрана (шторки сверху и снизу)
  - Визуальный эффект "обмана" — урон кажется реальным, но после затемнения здоровье возвращается
  - Уведомления о том, что это была иллюзия
- Обновленные особенности карт (все 13 карт с характеристиками)
- Новые иконки: ic_skull.xml (череп), ic_dream.xml (облачко/звездочка), ic_illusion.xml (глаз с эффектом)

### Fixed
- Эффект "Сын депутата" — теперь срабатывает только один раз за бой
- Пропуск хода при оглушении — корректная обработка, если все карты оглушены
- Логика щита — урон правильно распределяется между щитом и здоровьем
- Анимация атаки — синхронизация с эффектами особенностей

## [v1.1.0] - 2026-03-30

### Added
- Уникальные особенности карт (10 карт):
  - Горохострел (id=1): с вероятностью 20% атакует повторно
  - Древний сожитель (id=2): с вероятностью 20% все карты атакуют повторно
  - Племя потерянных (id=3): с вероятностью 35% оглушает случайную карту палкой (пропускает ход)
  - Красный дьявол (id=4): с вероятностью 50% атака x2, но защита -5
  - Сын депутата (id=5): с вероятностью 80% игнорирует 50% урона, но пропускает ход
  - Мини Пекка (id=6): с вероятностью 65% атака +10, но защита -5
  - Мастер-класс (id=7): с вероятностью 80% восстанавливает 15 здоровья, но пропускает ход
  - Единение с природой (id=8): с вероятностью 50% +10 к атаке на весь бой, но 40% шанс получить -5 к защите и здоровью
  - Роланд Азер (id=9): с вероятностью 60% увеличивает урон всем картам на 39%
  - Чай (id=10): с вероятностью 65% атака +18, но 60% шанс задеть своих
- Система особенностей:
  - Отдельный файл CardSpecialEffect.kt для управления всеми эффектами
  - Визуальные иконки для каждой особенности в едином стиле
  - Множественные иконки на карте (до 3 штук) с разными позициями
  - Всплывающие подсказки при нажатии на иконку особенности
- Профессиональная анимация атаки:
  - Полет атакующей карты к цели (по диагонали)
  - Увеличение карты во время полета
  - Вздрагивание цели (3-ступенчатая анимация)
  - Красная подсветка цели на момент удара
  - Возврат карты с эффектом отскока
  - Блокировка кликов во время анимации
- Система оглушения:
  - Визуальное выделение оглушенной карты (темный оверлей + иконка молнии)
  - Невозможность выбора оглушенной карты для атаки
  - Автоматический пропуск хода, если все карты оглушены
  - Снятие оглушения в начале хода
- Промокоды (обновленные): 31.03.2026 (анимация разноцветных шаров), hack (активация хак-режима, статистика 999, не обновляется), normal (выход из хак-режима, сброс статистики в 0), reset (полный сброс игры до заводских настроек), comment (диалог с отзывами игроков в профессиональном стиле)
- Титры: кликабельная ссылка на GitHub с иконкой 🐙, стрелка-индикатор для кликабельных элементов, единый стиль для всех элементов титров
- Полноэкранный режим (скрыта верхняя панель) во всех активностях
- Улучшенный инвентарь — фото слева, иконки особенностей справа
- Адаптивные размеры карточек в горизонтальном режиме
- Иконки в едином стиле с золотым контуром

### Fixed
- Критические ошибки:
  - Бессмертие карт — карты теперь правильно умирают при health <= 0
  - Отрицательное здоровье — здоровье не может стать меньше 0
  - Логика щита — сначала урон идет в щит, остаток в здоровье
  - Оглушение "Сына депутата" — срабатывает только один раз за бой
  - Пропуск хода при оглушении — если все карты оглушены, ход пропускается
  - Дублирование переменных в GameActivity
  - Конфликт ресурсов themes.xml и styles.xml
- Графические ошибки:
  - Исправлены цвета редкостей
  - Исправлено наложение текста в горизонтальном режиме
  - Исправлена позиция иконок особенностей
  - Убран белый прямоугольник при нажатии на ссылку GitHub

## [v1.0.0] - 2026-03-20

### Added
- 25 уникальных карт с индивидуальными характеристиками
- 6 уровней редкости: Обычный, Редкий, Сверхредкий, Эпический, Мифический, Легендарный
- Система цветов редкости: Редкий — зелёный (#2ECC71), Сверхредкий — синий (#3498DB), Мифический — красный (#E74C3C), Легендарный — золотой (#F1C40F)
- Система категорий карт: Колода, Карты, Недоступно
- Лимит колоды: 3 карты
- Сохранение состояния колоды через SharedPreferences
- Случайный выбор первого хода (игрок или ИИ)
- Полноценная боевая система с учётом здоровья, урона и защиты
- Атака: урон = (атака атакующего) - (защита цели), при пробитии защиты — урон по здоровью
- ИИ противник: случайный выбор карты для атаки, случайный выбор цели
- Подсветка выбранной карты с затемнением остальных
- Индикатор хода (золотой — игрок, красный — ИИ)
- Подсчёт побед и поражений
- Подсчёт технических поражений (выход из боя)
- Круговая диаграмма (три сектора: победы, поражения, технические поражения)
- Счётчик собранных карт (из доступных в инвентаре)
- Промокоды: DEVIL2024 (активирует демоническую форму), CARDMASTER (даёт легендарную карту), WELCOME (приветственный бонус 100 монет)
- Появление элементов с эффектами Bounce и Overshoot
- Анимация атаки (движение карты вперёд и назад)
- Анимация пульсации кнопки битвы
- Анимация "вырастания" чудика для карты "Красный дьявол"
- Анимация вращения кнопки обновления статистики
- Стеклянная карточка (glass morphism) на всех экранах
- Декоративные элементы с размытием
- Поддержка портретной и ландшафтной ориентации
- Полноэкранный иммерсивный режим
- Анимация наведения на кнопки
- Экран дисклеймера с текстом при запуске, плавное появление (заголовок → линия → текст), плавное исчезновение (все элементы одновременно), пропуск по нажатию на экран

### Fixed
- Критические ошибки:
  - Двойной запуск главного меню после дисклеймера при повороте экрана (отмена таймера при повороте и при пропуске)
  - Невозможность выбора цели после хода противника (корректная смена флага хода после атаки)
  - Обрезка текста в горизонтальном режиме (уменьшение размера шрифтов и отступов)
  - Сохранение состояния при повороте в GameActivity (onSaveInstanceState / onRestoreInstanceState)
- Синтаксические ошибки:
  - Исправлены импорты во всех файлах
  - Убраны неиспользуемые переменные
  - Исправлены типы в анимациях
